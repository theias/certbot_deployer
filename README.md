certbot_deployer
===========

Pluggable certbot deploy hook framework

Requires Python 3.9+

# Installation

You can install with [pip]:

```sh
python3 -m pip install certbot_deployer
```

Or install from source:

```sh
git clone <url>
pip install certbot_deployer
```
